GUIDES
-------------------------

### How to launch project
- Install Ruby, follow specific version at `.ruby-version`

- `bundle`

- Install Node, follow specific version at `.nvmrc`

- `yarn install`

- `rails s`

### How to run test cases

- Run Command: `rspec spec/services/command_service_spec.rb`
